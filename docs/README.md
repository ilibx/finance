# ERP 系统文档

## 概述

基于 Golang 实现的企业资源计划（ERP）管理系统，采用类似 Odoo 的功能架构，为企业提供全面的管理解决方案。

## 功能模块

### 1. 项目管理
- **项目创建与管理**：支持多项目并行管理
- **项目进度跟踪**：实时监控项目进展
- **项目成本核算**：精确计算项目成本

### 2. 财务对账
- **用户充值记录管理**：完整记录用户充值流水
- **供应商充值记录管理**：管理供应商预付款项
- **收支对账**：自动对账功能
- **财务报表**：生成各类财务报表

### 3. 发票管理
- **销售发票生成**：自动生成销售发票
- **采购发票管理**：管理采购环节发票
- **发票导入导出**：支持批量操作
- **发票状态跟踪**：实时追踪发票状态

### 4. 供应链管理
- **供应商管理**：维护供应商信息
- **采购订单管理**：全流程采购管理
- **库存管理**：实时库存监控
- **供应商充值与发票**：供应商资金管理

### 5. 销售管理
- **客户/用户管理**：客户关系管理
- **产品销售订单**：订单全生命周期管理
- **消费账单管理**：详细账单记录
- **销售发票生成**：自动化开票

## 技术架构

### 技术栈

| 组件 | 技术选型 | 版本要求 |
|------|----------|----------|
| 编程语言 | Golang | 1.19+ |

| Web 框架 | net/http (标准库) | - |

```
├─────────────────────────────────────┤
│         Service Layer               │
│     (internal/service/)             │
├─────────────────────────────────────┤
│       Repository Layer              │
│    (internal/repository/)           │
├─────────────────────────────────────┤
│          Database                   │

└─────────────────────────────────────┘
```

## 快速开始

### 环境要求

- **Go**: 1.19 或更高版本
- **PostgreSQL**: 12 或更高版本
- **操作系统**: Linux / macOS / Windows

### 步骤 1: 数据库初始化

```bash
# 创建数据库
createdb erp_db

# 执行初始化脚本
psql -d erp_db -f sql/init.sql
```

### 步骤 2: 配置环境变量

```bash
# 服务器配置
export SERVER_PORT=8080

# 数据库配置
export DB_HOST=localhost
export DB_PORT=5432
export DB_USER=postgres
export DB_PASSWORD=postgres
export DB_NAME=erp_db
```

### 步骤 3: 编译与运行

```bash
# 方式一：编译后运行
go build -o erp-system ./cmd/main.go
./erp-system

# 方式二：直接运行
go run ./cmd/main.go
```

### 步骤 4: 验证服务

服务启动后，访问 `http://localhost:8080` 验证系统是否正常运行。

## Excel 导入导出

### 支持的导入导出类型

1. **消费账单** - 批量导入/导出用户消费记录
2. **用户充值记录** - 批量导入用户充值数据
3. **供应商充值记录** - 批量导入供应商充值数据
4. **供应商发票** - 批量导入供应商发票信息
5. **销售发票** - 导出生成的销售发票

### Excel 文件格式规范

#### 消费账单导入格式

| 列名 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 用户 ID | Integer | 是 | 用户唯一标识 |
| 订单号 | String | 是 | 订单编号 |
| 金额 | Decimal | 是 | 消费金额 |
| 描述 | String | 否 | 消费描述 |
| 日期 | Date | 是 | 消费日期 (YYYY-MM-DD) |

**示例数据：**
```
| 用户 ID | 订单号 | 金额 | 描述 | 日期 |
|--------|--------|------|------|------|
| 1 | ORD20240101001 | 100.00 | 产品购买 | 2024-01-01 |
```

#### 用户充值记录导入格式

| 列名 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 用户 ID | Integer | 是 | 用户唯一标识 |
| 金额 | Decimal | 是 | 充值金额 |
| 支付方式 | String | 是 | 支付方式 (bank_transfer/alipay/wechat) |
| 备注 | String | 否 | 充值备注 |
| 日期 | Date | 是 | 充值日期 (YYYY-MM-DD) |

**示例数据：**
```
| 用户 ID | 金额 | 支付方式 | 备注 | 日期 |
|--------|------|----------|------|------|
| 1 | 1000.00 | bank_transfer | 银行转账 | 2024-01-01 |
```

#### 供应商充值记录导入格式

| 列名 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 供应商 ID | Integer | 是 | 供应商唯一标识 |
| 金额 | Decimal | 是 | 充值金额 |
| 支付方式 | String | 是 | 支付方式 |
| 备注 | String | 否 | 充值备注 |
| 日期 | Date | 是 | 充值日期 (YYYY-MM-DD) |

**示例数据：**
```
| 供应商 ID | 金额 | 支付方式 | 备注 | 日期 |
|----------|------|----------|------|------|
| 1 | 5000.00 | bank_transfer | 预付款 | 2024-01-01 |
```

#### 供应商发票导入格式

| 列名 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 供应商 ID | Integer | 是 | 供应商唯一标识 |
| 发票号 | String | 是 | 发票编号 |
| 金额 | Decimal | 是 | 发票金额 |
| 税额 | Decimal | 是 | 税额 |
| 日期 | Date | 是 | 开票日期 (YYYY-MM-DD) |

**示例数据：**
```
| 供应商 ID | 发票号 | 金额 | 税额 | 日期 |
|----------|--------|------|------|------|
| 1 | INV001 | 10000.00 | 1300.00 | 2024-01-01 |
```

## 常见问题

### 数据库连接失败

请检查：
1. PostgreSQL 服务是否已启动
2. 环境变量配置是否正确
3. 数据库用户权限是否充足

### Excel 导入失败

请确认：
1. Excel 文件格式是否符合规范
2. 必填字段是否完整
3. 数据格式是否正确（特别是日期和金额）
